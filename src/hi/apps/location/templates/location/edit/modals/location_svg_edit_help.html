Quick Reference Guide
{% icon "move" color="primary" %}
General
  • Drag from Palette Drop onto canvas to add element
  • Click & Drag Pan the canvas
  • Mouse Wheel / + / - Zoom in/out
  • CTRL + Z Undo last change
  • SHIFT + Drag Fine positioning (bypass snap)
  • Snap (footer) Set grid size, 0 to disable
{% icon "zoom" color="primary" %}
Icon Editing
  • Click Select icon
  • Drag Move icon
  • S + Drag/Wheel/+/- Scale icon
  • R + Drag/Wheel/+/- Rotate icon
  • M Mirror horizontally
  • X / DEL Delete selected icon
  • ESC Deselect
{% icon "path" color="primary" %}
Path Editing
  • Click Select path to edit vertices
  • Drag Point Move single vertex
  • CTRL + Drag Move entire path
  • I / INSERT Insert vertex
  • X / DEL Delete vertex
  • A / + Add new path segment
  • CTRL + X Delete entire path
  • ESC Finish path editing
