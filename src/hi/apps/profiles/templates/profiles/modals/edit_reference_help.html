Quick Reference Guide
{% icon "move" size="sm" color="primary" %}
General
  • Click & Drag Pan the view
  • Mouse Wheel / + / - Zoom in/out
  • CTRL+S + Drag/Wheel Scale the view
  • CTRL+R + Drag/Wheel Rotate the view
  • SHIFT + Drag Fine positioning (bypass snap)
  • Snap (footer) Set grid size, 0 to disable
{% icon "zoom" size="sm" color="primary" %}
Icon Editing
  • Click Select entity
  • Drag Move entity
  • S + Drag/Wheel/+/- Scale entity
  • R + Drag/Wheel/+/- Rotate entity
  • ESC Deselect
{% icon "path" size="sm" color="primary" %}
Path Editing
  • Click Path Select path to edit vertices
  • Click Canvas Extend path (add vertex)
  • Drag Point Move single vertex
  • CTRL + Drag Move entire path
  • I / INSERT Insert vertex
  • X / DEL Delete vertex
  • A / + Add new segment
  • ESC Finish path editing
{% icon "info-circle" size="sm" color="secondary" %}
Important Note

View pan/zoom/rotate changes are only saved when the "View" editing pane is showing.
