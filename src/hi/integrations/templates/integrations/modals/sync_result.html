{% extends "modals/action_3.html" %} {% load icons %} {% load static %} {% comment %} Integration sync result modal. The single end-of-sync surface: always rendered after sync completes, regardless of what changed. Updates and removes are surfaced here in their own right rather than being silently swallowed by an automatic transition to the placement. Renders a hero headline, a stat-card row of change counts, color-coded per-category enumerations of affected entities, and collapsible Errors / Details sections. When the sync produced newly-created entities, the footer's primary CTA navigates to the placement modal (via antinode modal-to-modal); otherwise the footer is a single dismissal button. The modal title bar keeps the generic HI logo + short generic title ('Connect Result' / 'Update Check Result'); the integration's identity is surfaced as a body subtitle row (logo + label) where the white background gives the transparent integration logos predictable contrast. Context: sync_result : IntegrationSyncResult (required) integration_data : IntegrationData (required) placement_url : str (optional) — target of the 'Place N items' CTA.
